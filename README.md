This project demonstrates Simple Linear Regression using Python and Scikit-learn.
# Simple Linear Regression Project

## Objective
To predict y based on x using Linear Regression.

## Steps
- Loaded dataset
- Trained Linear Regression model
- Predicted values
- Plotted regression line
- Evaluated using R² score

## Tools Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## Result
Model shows positive linear relationship with good R² score.
